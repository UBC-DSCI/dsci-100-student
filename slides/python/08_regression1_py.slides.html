1. Is our model any good?

Suppose the blue line depicts our predictions from k-nn regression.

The red lines depict the error in our predictions, i.e., the difference between the $i^\text{th}$ test data response and our prediction $\hat{y}_i$:

  • small datasets: the nearest neighbours are the same and thus the mean in a certain area is the same therefore we have flat lines

1. Is our model any good?

We (roughly) add up these errors to evaluate our regression model

  • Not out of 1, but instead in units of the target variable (bit harder to interpret)

Root Mean Squared Prediction Error (RMSPE):

$$RMSPE = \sqrt{\frac{1}{n}\sum\limits_{i=1}^{n}(y_i - \hat{y_i})^2}$$

- $n$ is the number of observations

- $y_i$ is the observed value for the $i^\text{th}$ test observation

- $\hat{y_i}$ is the forcasted/predicted value for the $i^\text{th}$ test observation

RMSE vs RMSPE

  • Root Mean Squared Error (RMSE): The same formula except on the data used to train the model
    • this indicates how well our model can fit our data
  • Root Mean Squared Prediction Error (RMSPE): predicting on unseen data.
    • this indicates how well our model generalizes to future data

This distinction is very important, but no standard terminology.

  • Prices tend to be +/- 90,000 off that line

Final model from k-nn regression

For this model, RMSPE is roughly $83,000. How can we interpret this?

Our final models test error as assessed by RMSPE is 83,825. Note that RMSPE is measured in the same units as the response variable. In other words, on new observations, we expect the error in our prediction to be roughly 83,825. From one perspective, this is good news: this is about the same as the cross-validation RMSPE estimate of our tuned model (which was 86,231, so we can say that the model appears to generalize well to new data that it has never seen before. However, much like in the case of KNN classification, whether this value for RMSPE is goodi.e., whether an error of around 83,825 is acceptabledepends entirely on the application. In this application, this error is not prohibitively large, but it is not negligible either; $83,825 might represent a substantial fraction of a home buyers budget, and could make or break whether or not they could afford put an offer on a house.
